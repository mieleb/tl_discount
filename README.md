## Prerequisites

## Docksal

Install docksal [documentation](https://docs.docksal.io/getting-started/setup/) 

## Setup

After installing docksal.
Go to the root of the project in terminal.

run `fin project start`
run `fin init`

## Tests

run `fin phpunit`
